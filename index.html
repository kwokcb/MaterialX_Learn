MaterialX Learn

You have reached the home of "MaterialX Learn".

The intent of this site is to provide learning materials as well as complementary utilities for working with MaterialX

This site is up-to-date to the 1.39 release of MaterialX

"End of Line..."

It's been an interesting journey but due to lack of interest this site be retiring soon.

This site is sync'd to the official 1.39 release of MaterialX release. For the short-term please log a Github request if there is a desire for additional sponsored work. Read Github repository announcements for further information.

Thanks for all those that have shown interest in this site. Bernard and Matthew Kwok (July, 2024).

"The Digital Frontier..."

Procedural pattern / shader generation using MaterialX graphs is the current focus. A supporting Web-based shader graph editor is now publicly available here. A video demonstrating some features can be found here

Beta versions of web based tools are accessible under the "Utilities" menu until. They will be migrated out by the fall of 2024.

  • Graph Editor with Plug-in Renderer
  • Shader Generation / Reflection utilities
