Keyboard Shortcuts
Navigation
  • F - Fit to view (current)
  • G - Fit all to view
  • / - Focus search
Node Navigation
  • or W - Navigate to parent
  • or S - Navigate to first child
  • or A - Previous sibling
  • or D - Next sibling
  • Home - First sibling
  • End - Last sibling
Tree Controls
  • E - Expand all
  • C - Collapse all
  • O - Toggle orientation
  • T - Toggle theme
Mouse Controls
  • Click node: Expand/collapse or select
  • Click label: Same as clicking node
  • Drag: Pan the tree
  • Scroll: Zoom in/out
